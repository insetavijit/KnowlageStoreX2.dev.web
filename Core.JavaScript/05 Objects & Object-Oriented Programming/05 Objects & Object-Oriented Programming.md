# **[[05 Objects & Object-Oriented Programming]]**

Master JavaScript objects and OOP principles. Learn object creation, methods, prototypes, and classes. Build object-oriented applications. Objects are the foundation of JavaScript's programming paradigm.

|Topic|Focus & Purpose|
|---|---|
|**[[5.1 Object Basics]]**|Object literals; properties; methods; accessing properties; dot notation; bracket notation; object structure. Object fundamentals.|
|**[[5.2 Object Methods]]**|Defining methods; this keyword; method context; computed properties; shorthand syntax; method patterns. Object behavior.|
|**[[5.3 Object Manipulation]]**|Adding properties; deleting properties; Object.keys; Object.values; Object.entries; property enumeration. Working with objects.|
|**[[5.4 Object Destructuring]]**|Destructuring assignment; nested destructuring; default values; renaming; rest in objects; practical patterns. Modern syntax.|
|**[[5.5 Constructor Functions]]**|Constructor pattern; new keyword; this in constructors; creating instances; constructor conventions. Object creation pattern.|
|**[[5.6 Prototypes & Inheritance]]**|Prototype chain; prototype property; **proto**; Object.create; prototypal inheritance; prototype methods. Inheritance mechanism.|
|**[[5.7 ES6 Classes]]**|class syntax; constructor; methods; static methods; class expressions; class vs constructor; modern OOP. Class-based OOP.|
|**[[5.8 Project: Bank Account System]]**|Building account management; classes; encapsulation; methods; object interaction; OOP principles. Real OOP application.|
