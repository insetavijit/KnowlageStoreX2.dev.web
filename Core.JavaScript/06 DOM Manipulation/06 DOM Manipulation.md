# **[[06 DOM Manipulation]]**

Master the Document Object Model for interactive web pages. Learn selecting elements, modifying content, and dynamic HTML generation. Build interactive user interfaces. DOM manipulation is essential for creating dynamic web applications.

|Topic|Focus & Purpose|
|---|---|
|**[[6.1 DOM Basics]]**|What is DOM; DOM tree; nodes vs elements; document object; DOM structure; browser rendering. Understanding DOM.|
|**[[6.2 Selecting Elements]]**|getElementById; querySelector; querySelectorAll; getElementsByClassName; getElementsByTagName; NodeList vs HTMLCollection. Finding elements.|
|**[[6.3 Modifying Content]]**|textContent; innerHTML; innerText; creating text nodes; updating content; XSS considerations. Changing content.|
|**[[6.4 Modifying Attributes]]**|getAttribute; setAttribute; removeAttribute; dataset; class manipulation; style property. Element attributes.|
|**[[6.5 Creating & Removing Elements]]**|createElement; appendChild; insertBefore; removeChild; cloneNode; replacing elements. Dynamic DOM.|
|**[[6.6 CSS Class Manipulation]]**|classList.add; classList.remove; classList.toggle; classList.contains; className; styling elements. Dynamic styling.|
|**[[6.7 DOM Traversal]]**|parentNode; children; firstChild; lastChild; nextSibling; previousSibling; navigating DOM. Walking the tree.|
|**[[6.8 Project: Dynamic Content App]]**|Building dynamic webpage; DOM manipulation; creating elements; user input; real-time updates. Interactive application.|
