# **[[13 Regular Expressions]]**

Master regular expressions for pattern matching and text processing. Learn regex syntax, methods, and practical applications. Build text processing tools. Regex is essential for validation, searching, and data extraction.

|Topic|Focus & Purpose|
|---|---|
|**[[13.1 Regex Basics]]**|Regex syntax; literal characters; creating regex; test method; exec method; regex flags. Pattern matching fundamentals.|
|**[[13.2 Character Classes]]**|Character sets; ranges; negation; predefined classes (\d, \w, \s); escaping; special characters. Pattern building.|
|**[[13.3 Quantifiers]]**|*, +, ?, {n}, {n,m}; greedy vs lazy; repetition; optional patterns; quantifier patterns. Repetition control.|
|**[[13.4 Anchors & Boundaries]]**|^ start; $ end; \b word boundary; \B; multiline mode; position matching. Position patterns.|
|**[[13.5 Groups & Capturing]]**|Capturing groups; non-capturing groups; backreferences; named groups; group patterns. Grouping patterns.|
|**[[13.6 String Methods with Regex]]**|match; replace; search; split; matchAll; string methods. Regex integration.|
|**[[13.7 Practical Regex Patterns]]**|Email validation; phone validation; URL matching; password strength; common patterns. Real-world regex.|
|**[[13.8 Project: Form Validator]]**|Building validation system; regex patterns; email validation; password requirements; user feedback. Production validator.|
