# **[[3.2 Parameters & Arguments]]**

## Overview

This topic covers function parameters and arguments. You'll learn the difference between parameters and arguments, default parameter values, rest parameters for variable arguments, the arguments object, and parameter destructuring.

## Parameters vs Arguments

Parameters are placeholders; arguments are actual values passed.

**Definitions:**
```javascript
// Parameters: variables in function definition
function greet(name, greeting) {
    console.log(greeting + ", " + name);
}

// Arguments: values passed when calling
greet("Alice", "Hello");  // "Alice" and "Hello" are arguments
```

**Multiple Parameters:**
```javascript
function calculate(a, b, operation) {
    switch (operation) {
        case "add": return a + b;
        case "subtract": return a - b;
        case "multiply": return a * b;
        case "divide": return b !== 0 ? a / b : null;
    }
}

calculate(10, 5, "add");      // 15
calculate(10, 5, "multiply"); // 50
```

**Missing Arguments:**
```javascript
function greet(name, greeting) {
    console.log(greeting + ", " + name);
}

greet("Alice");  // "undefined, Alice"
greet();         // "undefined, undefined"
```

**Extra Arguments:**
```javascript
function add(a, b) {
    return a + b;
}

add(1, 2, 3, 4, 5);  // Returns 3 (only uses first two)
```

## Default Parameters

Provide fallback values for missing arguments.

**Syntax:**
```javascript
function greet(name = "Guest", greeting = "Hello") {
    console.log(`${greeting}, ${name}!`);
}

greet();                 // "Hello, Guest!"
greet("Alice");          // "Hello, Alice!"
greet("Bob", "Hi");      // "Hi, Bob!"
greet(undefined, "Hey"); // "Hey, Guest!"
```

**Expressions as Defaults:**
```javascript
function createId(prefix = "id", num = Date.now()) {
    return `${prefix}_${num}`;
}

createId();           // "id_1704123456789"
createId("user");     // "user_1704123456789"
createId("item", 1);  // "item_1"
```

**Function Calls as Defaults:**
```javascript
function getDefaultName() {
    return "Anonymous";
}

function greet(name = getDefaultName()) {
    console.log("Hello, " + name);
}

greet();        // "Hello, Anonymous"
greet("Alice"); // "Hello, Alice"
```

**Parameter Reference:**
```javascript
function createRange(start, end = start + 10) {
    return { start, end };
}

createRange(5);     // { start: 5, end: 15 }
createRange(5, 20); // { start: 5, end: 20 }
```

## Rest Parameters

Collect remaining arguments into an array.

**Syntax:**
```javascript
function sum(...numbers) {
    let total = 0;
    for (let num of numbers) {
        total += num;
    }
    return total;
}

sum(1, 2);           // 3
sum(1, 2, 3, 4, 5);  // 15
sum();               // 0
```

**With Regular Parameters:**
```javascript
function greetAll(greeting, ...names) {
    for (let name of names) {
        console.log(`${greeting}, ${name}!`);
    }
}

greetAll("Hello", "Alice", "Bob", "Charlie");
// "Hello, Alice!"
// "Hello, Bob!"
// "Hello, Charlie!"
```

**Must Be Last Parameter:**
```javascript
// Valid
function valid(first, second, ...rest) { }

// Invalid - rest must be last
// function invalid(...rest, last) { }  // SyntaxError
```

**Practical Examples:**
```javascript
function max(...values) {
    if (values.length === 0) return undefined;
    return Math.max(...values);
}

function joinPaths(base, ...paths) {
    return [base, ...paths].join('/');
}

joinPaths("home", "user", "documents");  // "home/user/documents"
```

## Arguments Object

Legacy way to access all arguments (avoid in modern code).

**Basic Usage:**
```javascript
function oldSum() {
    let total = 0;
    for (let i = 0; i < arguments.length; i++) {
        total += arguments[i];
    }
    return total;
}

oldSum(1, 2, 3, 4);  // 10
```

**Arguments Properties:**
```javascript
function showArgs() {
    console.log(arguments.length);  // Number of arguments
    console.log(arguments[0]);       // First argument
    console.log(arguments[1]);       // Second argument
}

showArgs("a", "b", "c");  // 3, "a", "b"
```

**Not a Real Array:**
```javascript
function demo() {
    // arguments is array-like, not a real array
    // arguments.map();  // Error!
    
    // Convert to array
    let args = Array.from(arguments);
    // Or
    let args2 = [...arguments];
    
    return args.map(x => x * 2);
}
```

**Rest Parameters vs Arguments:**
```javascript
// Prefer rest parameters (modern)
function modern(...args) {
    return args.map(x => x * 2);  // Works directly
}

// Arguments object (legacy)
function legacy() {
    let args = [...arguments];
    return args.map(x => x * 2);  // Needs conversion
}
```

## Parameter Destructuring

Extract values from objects/arrays in parameters.

**Object Destructuring:**
```javascript
function createUser({ name, email, age = 18 }) {
    return {
        name,
        email,
        age,
        createdAt: Date.now()
    };
}

createUser({ name: "John", email: "john@example.com" });
// { name: "John", email: "john@example.com", age: 18, createdAt: ... }
```

**Array Destructuring:**
```javascript
function getPoint([x, y]) {
    return { x, y };
}

getPoint([10, 20]);  // { x: 10, y: 20 }

function sumFirstTwo([a, b, ...rest]) {
    console.log("Rest:", rest);
    return a + b;
}

sumFirstTwo([1, 2, 3, 4, 5]);  // Rest: [3, 4, 5], returns 3
```

**Nested Destructuring:**
```javascript
function processUser({ name, address: { city, country = "USA" } }) {
    return `${name} from ${city}, ${country}`;
}

processUser({
    name: "Alice",
    address: { city: "New York" }
});  // "Alice from New York, USA"
```

**With Default Object:**
```javascript
function configure({ timeout = 3000, retries = 3 } = {}) {
    return { timeout, retries };
}

configure();                    // { timeout: 3000, retries: 3 }
configure({ timeout: 5000 });   // { timeout: 5000, retries: 3 }
configure({ retries: 5 });      // { timeout: 3000, retries: 5 }
```

## Summary / Key Takeaways

- **Parameters:** Variables in function definition; **Arguments:** Actual values passed
- **Default parameters:** `function(x = default)` provides fallback values
- **Rest parameters:** `...args` collects remaining arguments into array; must be last
- **Arguments object:** Legacy array-like object; prefer rest parameters
- **Destructuring:** Extract object/array properties directly in parameter list
- **Default for destructuring:** `{ prop = default }` or `= {}` for entire parameter
- **Best practices:** Use rest over arguments, destructure for config objects

Parameters and arguments enable flexible, reusable functions with clean interfaces.
