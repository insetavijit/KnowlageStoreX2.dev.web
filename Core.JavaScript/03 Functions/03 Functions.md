# **[[03 Functions]]**

Master functions as reusable code blocks. Learn function declarations, expressions, arrow functions, and parameters. Build modular, maintainable code. Functions are the building blocks of JavaScript programs and enable code reusability.

|---|---|

|**[[3.2 Parameters & Arguments]]**|Parameters; arguments; default parameters; rest parameters; argument object; parameter destructuring. Input handling.|
|**[[3.3 Function Expressions]]**|Function expressions; anonymous functions; named function expressions; assigning functions; expression vs declaration. Flexible syntax.|
|**[[3.4 Arrow Functions]]**|Arrow function syntax; implicit return; no binding of this; when to use arrows; concise syntax; limitations. Modern functions.|
|**[[3.5 Scope]]**|Global scope; function scope; block scope; lexical scope; scope chain; variable shadowing; scope rules. Variable visibility.|
|**[[3.6 Closures]]**|Closure concept; lexical environment; private variables; closure use cases; memory considerations; practical patterns. Advanced scope.|
|**[[3.7 Higher-Order Functions]]**|Functions as arguments; returning functions; callbacks; function composition; functional programming basics. Advanced patterns.|
|**[[3.8 Project: To-Do List Functions]]**|Building to-do functions; CRUD operations; function modularity; parameter handling; return values. Modular code.|
