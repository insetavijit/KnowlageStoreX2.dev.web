# **[[08 Asynchronous JavaScript]]**

Master asynchronous programming for handling time-dependent operations. Learn callbacks, promises, async/await, and fetch API. Build applications that communicate with servers. Async JavaScript is essential for modern web development.

|Topic|Focus & Purpose|
|---|---|
|**[[8.1 Synchronous vs Asynchronous]]**|Execution models; blocking vs non-blocking; call stack; event loop; concurrency model. Understanding async.|
|**[[8.2 Callbacks]]**|Callback functions; callback pattern; callback hell; error-first callbacks; callback limitations. Traditional async.|
|**[[8.3 Promises]]**|Promise concept; creating promises; then/catch; promise states; promise chaining; error handling. Modern async.|
|**[[8.4 Async/Await]]**|async keyword; await keyword; async functions; error handling with try-catch; async patterns. Cleaner async code.|
|**[[8.5 Fetch API]]**|Fetching data; HTTP requests; response handling; JSON; headers; error handling; REST APIs. API communication.|
|**[[8.6 Promise Methods]]**|Promise.all; Promise.race; Promise.allSettled; Promise.any; parallel execution; promise utilities. Advanced promises.|
|**[[8.7 Error Handling Async]]**|try-catch in async; promise rejection; error propagation; graceful degradation; retry logic. Robust async code.|
|**[[8.8 Project: Weather App]]**|Building weather application; fetch API; async/await; API integration; error handling; dynamic UI. Real API project.|
