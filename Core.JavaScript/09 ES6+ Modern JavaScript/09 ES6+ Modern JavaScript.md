# **[[09 ES6+ Modern JavaScript]]**

Master modern JavaScript features from ES6 onwards. Learn destructuring, template literals, modules, and new syntax. Write clean, modern code. ES6+ features dramatically improve code quality and developer experience.

|Topic|Focus & Purpose|
|---|---|
|**[[9.1 Let, Const & Var]]**|Block scoping; const immutability; let vs var; temporal dead zone; hoisting differences; choosing declaration. Modern variables.|
|**[[9.2 Template Literals]]**|Backtick syntax; string interpolation; multiline strings; expressions in strings; tagged templates. Modern strings.|
|**[[9.3 Destructuring]]**|Array destructuring; object destructuring; nested destructuring; default values; rest in destructuring. Clean syntax.|
|**[[9.4 Spread & Rest]]**|Spread operator; rest parameters; array spreading; object spreading; practical use cases. Flexible syntax.|
|**[[9.5 Default Parameters]]**|Function default values; default expressions; default with destructuring; parameter defaults. Better functions.|
|**[[9.6 Enhanced Object Literals]]**|Property shorthand; method shorthand; computed property names; object enhancements. Cleaner objects.|
|**[[9.7 Modules]]**|ES6 modules; import/export; named exports; default exports; module syntax; module scope. Code organization.|
|**[[9.8 Project: Modern App Setup]]**|Building with ES6+ features; modules; modern syntax; clean code; best practices. Modern development.|
