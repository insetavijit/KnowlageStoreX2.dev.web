# **[[04 Arrays & Data Structures]]**

Master arrays and data manipulation. Learn array methods, iteration, transformation, and searching. Build data-driven applications. Arrays are fundamental for managing collections of data in JavaScript.

| Topic                                   | Focus & Purpose                                                                                                                       |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **[[4.1 Array Basics]]**                | Creating arrays; array literals; accessing elements; array length; index; array vs object; array characteristics. Array fundamentals. |
| **[[4.2 Array Methods Mutating]]**     | push, pop, shift, unshift; splice; reverse; sort; fill; modifying arrays; mutation. Changing arrays.                                  |
| **[[4.3 Array Methods Non-Mutating]]** | slice; concat; join; indexOf; includes; immutable operations; returning new arrays. Safe operations.                                  |
| **[[4.4 Array Iteration]]**             | forEach; for...of loop; traditional for; iterating arrays; accessing elements; loop patterns. Traversing arrays.                      |
| **[[4.5 Array Transformation]]**        | map; filter; reduce; chaining methods; functional approach; transforming data. Data manipulation.                                     |
| **[[4.6 Array Searching & Testing]]**   | find; findIndex; some; every; searching arrays; testing conditions; boolean checks. Finding data.                                     |
| **[[4.7 Spread & Rest Operators]]**     | Spread syntax (...); rest parameters; array copying; merging arrays; function arguments. Modern syntax.                               |
| **[[4.8 Project: Data Dashboard]]**     | Building data visualization; array methods; filtering; sorting; transforming data; statistics calculation. Real-world data.           |
