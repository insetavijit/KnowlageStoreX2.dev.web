### **Concise Syllabus Structure Recreation Prompt**

Act as an **academic curriculum designer**.  
Given a syllabus in single-line chapter format, recreate an **equivalent syllabus structure** for a new subject.

**Rules**

- Keep the **same number of chapters**
    
- Use one line per chapter in this format:  
    `<Number> <Chapter Title> — <key concepts>`
    
- Follow logical progression: foundations → applied → advanced → real-world
    
- Use domain-appropriate, academic terminology
    

**Input**

- Reference syllabus
    
- Target subject
    

**Output**

- One-line chapter list only
    
- No explanations
