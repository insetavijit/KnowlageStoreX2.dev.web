|**2.1.6 Removing List Styles**|Cover restyling for navigation|`list-style: none`, Resetting defaults|The first step to making a navbar is removing the bullets.|
|**[[2.1.6.1 The Property]]**|CSS|`list-style: none`|This single line removes the bullet points.|
|**[[2.1.6.2 Resetting Padding]]**|CSS|`padding: 0`|Removing bullets doesn't remove the indentation; set padding to zero.|
|**[[2.1.6.3 Resetting Margin]]**|CSS|`margin: 0`|Often you want to remove the default vertical spacing too.|
|**[[2.1.6.4 Flexbox/Grid]]**|Layout|`display: flex`|Turn the list horizontal by applying flexbox to the `<ul>`.|
|**[[2.1.6.5 Semantic Integrity]]**|Concept|Still a list|Even without bullets, it remains a semantic list to screen readers.|
|**[[2.1.6.6 Safari Issue]]**|Browser Quirk|VoiceOver behavior|Historically, removing styles sometimes stripped semantics in Safari (fixed in newer versions).|
