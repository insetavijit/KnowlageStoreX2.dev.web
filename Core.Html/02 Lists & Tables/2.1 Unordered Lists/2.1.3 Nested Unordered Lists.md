|**2.1.3 Nested Unordered Lists**|Detail hierarchical structures|Sub-lists, indented levels|To nest a list, put the new `<ul>` *inside* an `<li>` of the parent list.|
|**[[2.1.3.1 Correct Nesting]]**|Validity|Child of `<li>`, not `<ul>`|Wrong: `<ul>...<ul>...</ul></ul>`. Right: `<ul><li>...<ul>...</ul></li></ul>`.|
|**[[2.1.3.2 Visual Indentation]]**|Layout|Automatic padding|Browsers automatically indent nested lists further to the right.|
|**[[2.1.3.3 Marker Changes]]**|Defaults|Disc > Circle > Square|Browsers rotate marker styles for the first 3 levels of nesting.|
|**[[2.1.3.4 Use Cases]]**|Trees|Sitemaps, Multi-level menus|Use nesting to show parent-child relationships between items.|
|**[[2.1.3.5 Accessibility]]**|Structure|Level announcement|Screen readers announce nesting levels ("Level 2"), helping users visualize depth.|
|**[[2.1.3.6 Styling Depth]]**|Selectors|`ul ul`|Use descendant selectors to style nested lists differently.|
