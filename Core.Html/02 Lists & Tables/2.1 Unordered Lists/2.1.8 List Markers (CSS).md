|**Subtopic**|**Focus & Purpose**|**Key Concepts / Details**|**One-Line Recall**|
|---|---|---|---|
|**2.1.8 List Markers (CSS)**|Introduce customization|`::marker` pseudo-element, Custom colors/content|You can finally style the bullet point separately from the text.|
|**[[2.1.8.1 The ::marker]]**|Pseudo-element|Targeting the bullet|`li::marker { color: red; }` changes just the dot.|
|**[[2.1.8.2 Custom Content]]**|Replacement|`content: ">>"`|Replace the dot with any character or emoji.|
|**[[2.1.8.3 Limitation]]**|Restricted props|Font, Color, Content|You can't change margin/padding/background on a marker easily.|
|**[[2.1.8.4 Image Markers]]**|Alternative|`list-style-image`|Use an image file as the bullet (legacy method).|
|**[[2.1.8.5 Background Mock]]**|Workaround|Background image|For complex bullets, disable standard markers and use a background image on the `<li>`.|
|**[[2.1.8.6 Counters]]**|Advanced|`counter()`|Use CSS counters to create custom numbering inside markers.|
