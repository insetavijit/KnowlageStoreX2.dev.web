|**2.2.5 Value Attribute on LI**|Cover overriding specific items|Manual numbering, skips|Change the number of a specific item in the middle of a list.|
|**[[2.2.5.1 Syntax]]**|Per-item|`<li value="5">`|Forces this specific item to be number 5.|
|**[[2.2.5.2 Ripple Effect]]**|Sequence|Following items|The items *after* the value change will continue counting from the new number.|
|**[[2.2.5.3 Non-Sequential]]**|Gaps|1, 2, 10, 11|Useful for lists with missing steps or gaps.|
|**[[2.2.5.4 Deprecation Myth]]**|Status|Valid in HTML5|It was deprecated in HTML 4, but brought back in HTML5.|
|**[[2.2.5.5 Use Cases]]**|Legal|Outline numbering|Use it to handle complex legal outline skipping (e.g., Section 4, 5, 8).|
|**[[2.2.5.6 CSS Counters]]**|Alternative|More control|For complex logic, CSS counters are often more powerful.|
