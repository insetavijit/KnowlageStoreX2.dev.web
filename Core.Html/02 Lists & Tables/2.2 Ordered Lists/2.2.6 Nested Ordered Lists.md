|**2.2.6 Nested Ordered Lists**|Detail hierarchical numbering|Outlines, Sub-steps, Statutes|Use nested `<ol>` for outlines like 1, 1.1, 1.1.1 or I, A, 1.|
|**[[2.2.6.1 Nesting Pattern]]**|Syntax|Inside `<li>`|The nested `<ol>` must be a child of an `<li>`, not a child of the parent `<ol>`.|

|**[[2.2.6.3 Default Indentation]]**|Visuals|Padding|Browsers indent nested numbered lists just like unordered ones.|
|**[[2.2.6.4 Auto-Reset]]**|Counting|Scope|Each nested list starts counting from 1 (or its start value) independently.|
|**[[2.2.6.5 CSS Counters]]**|Advanced|`1.1` style|To get "1.1", "1.2" style numbering, you must use CSS counters.|
