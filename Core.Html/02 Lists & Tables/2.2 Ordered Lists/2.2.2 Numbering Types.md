|**2.2.2 Numbering Types**|Explain the `type` attribute|Decimals, Letters, Roman Numerals|Control *how* the browser counts (1, a, A, i, I).|
|**[[2.2.2.1 Default (1)]]**|Decimal|`type="1"`|1, 2, 3... (The default behavior).|
|**[[2.2.2.2 Lowercase Letters (a)]]**|Alpha|`type="a"`|a, b, c...|
|**[[2.2.2.3 Uppercase Letters (A)]]**|Alpha|`type="A"`|A, B, C...|
|**[[2.2.2.4 Roman Numerals (i/I)]]**|Classic|`type="i"` or `type="I"`|i, ii, iii... or I, II, III...|
|**[[2.2.2.5 CSS Alternative]]**|Style|`list-style-type`|Prefer CSS for purely visual changes; use attribute for semantic types (like legal docs).|
|**[[2.2.2.6 Limitation]]**|Scope|No "Step 1"|You can't add text like "Step" in the type attribute (need CSS counters for that).|
