|**2.6.8 Summary Attribute (Legacy)**|Address deprecated feature|History, Alternatives, Warning|The `summary` attribute is dead; long live the caption.|
|**[[2.6.8.1 Historical Purpose]]**|HTML 4|Hidden description|It was used to provide a hidden explanation of the table structure.|
|**[[2.6.8.2 Deprecation]]**|HTML5|Removal|Removed because authors abused it or put critical info in it that sighted users couldn't see.|
|**[[2.6.8.3 Modern Replacement]]**|Structure|`<caption>`|Use `<caption>` for titles.|
|**[[2.6.8.4 Complex Description]]**|Alternative|`aria-describedby`|Use `aria-describedby` pointing to a visible `<details>` block for complex explanations.|
|**[[2.6.8.5 Validation Error]]**|W3C|Flag|Using `summary` on a `<table>` will trigger an HTML5 validation error.|
|**[[2.6.8.6 Legacy Code]]**|Maintenance|Refactoring|If you see it in old code, move the text to a caption or helper paragraph.|
