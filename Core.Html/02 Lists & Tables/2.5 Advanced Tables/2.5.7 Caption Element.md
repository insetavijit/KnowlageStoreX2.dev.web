|**Subtopic**|**Focus & Purpose**|**Key Concepts / Details**|**One-Line Recall**|
|---|---|---|---|
|**2.5.7 Caption Element**|Detail table titles|Accessibility label, Visible title|The accessible title for the entire table.|
|**[[2.5.7.1 Significance]]**|Accessibility|The "Name"|The most important tag for making a table accessible; it tells screen readers what the table is.|
|**[[2.5.7.2 Placement]]**|First Rule|Top|Must be the *very first* child of the `<table>` element, before anything else.|
|**[[2.5.7.3 Default Style]]**|Visuals|Centered|Renders centered above the table by default, but you can move it with CSS.|
|**[[2.5.7.4 Comparison]]**|vs Heading|Association|Unlike an `<h2>` above a table, a `<caption>` is programmatically attached to the table.|
|**[[2.5.7.5 Description]]**|Content|Summary|Should briefly describe the data, e.g., "Monthly Sales Figures 2023".|
|**[[2.5.7.6 Side-Caption]]**|CSS|`caption-side`|You can move the caption to the bottom with `caption-side: bottom`.|
