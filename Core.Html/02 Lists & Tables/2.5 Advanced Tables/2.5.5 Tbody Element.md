|**2.5.5 Tbody Element**|Introduce body grouping|Semantic body, Scrollable areas|Groups the main data content of the table.|
|**[[2.5.5.1 Implicit Default]]**|Browser|Auto-insertion|Even if you don't write it, the browser wraps your `<tr>`s in a `<tbody>` automatically in the DOM.|
|**[[2.5.5.2 Multiple Bodies]]**|Structure|Sectioning|You can have multiple `<tbody>` elements in one table to group chunks of data.|
|**[[2.5.5.3 Styling]]**|Separation|Borders|You can put a thick border between `<tbody>` groups.|
|**[[2.5.5.4 Scrolling]]**|UX|Overflow|You can set `tbody { overflow-y: scroll }` (with some display tweaks) to make just the body scroll.|
|**[[2.5.5.5 Zoning]]**|Logic|Data vs Meta|Separates the data (body) from the labels (head) and summaries (foot).|
|**[[2.5.5.6 Selection]]**|JS|Targeting|Makes it easier to clear or update just the data rows with JavaScript.|
