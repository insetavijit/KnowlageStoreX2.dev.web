|---|---|---|---|
|**2.7.7 Alignment and Padding**|Readability micro-typography|`text-align`, `vertical-align`, `padding`|The difference between a messy grid and a professional report.|
|**[[2.7.7.1 Text Alignment]]**|Convention|Type-based|Text left, Numbers right, Headers align with data.|
|**[[2.7.7.2 Vertical Alignment]]**|Position|`vertical-align`|Usually `middle` or `top` (prevent short text from floating in the center of a tall row).|
|**[[2.7.7.3 Padding]]**|Breathing Room|`1rem`|Generous padding prevents the "Excel 95" look; give data space.|
|**[[2.7.7.4 Consistent Units]]**|System|`rem`/`em`|Use relative units for padding so it scales with text size.|
|**[[2.7.7.5 Char Alignment]]**|Advanced|Tabular nums|Use `font-variant-numeric: tabular-nums` so numbers stack perfectly.|
|**[[2.7.7.6 Header Match]]**|Rule|Consistency|Header alignment must match column alignment (don't center a header over right-aligned numbers).|
