|**Subtopic**|**Focus & Purpose**|**Key Concepts / Details**|**One-Line Recall**|
|---|---|---|---|
|**2.4.6 Column Headers**|Cover horizontal headers|Top row context, Scope col|Column headers sit at the top and describe the data below them.|
|**[[2.4.6.1 Position]]**|First Row|`<tr>`|Usually placed within the very first `<tr>` of the table (or `<thead>`).|
|**[[2.4.6.2 Scope]]**|Accessibility|`scope="col"`|Explicitly states: "I apply to everything in this vertical column".|
|**[[2.4.6.3 Styling]]**|Visuals|Visual Anchor|Often styled with a background color to separate them from data.|
|**[[2.4.6.4 Sticky Stickiness]]**|CSS|`position: sticky`|Keep them visible while scrolling long tables.|
|**[[2.4.6.5 Sorting]]**|Interaction|Clickable|Often used as the UI control for sorting the table data.|
|**[[2.4.6.6 Multi-Level]]**|Grouping|Colspan|You can have headers that span multiple sub-columns (e.g., "Q1" over "Jan", "Feb", "Mar").|
