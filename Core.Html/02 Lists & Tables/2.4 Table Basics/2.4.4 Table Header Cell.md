|**2.4.4 Table Header Cell**|Explain the `<th>` element|Labels, Scope, Semantics|Use `<th>` for the headings describing the data, not `<td>`.|

|**[[2.4.4.2 Default Style]]**|Visuals|Centered, Bold|Browsers make headers bold and centered by default.|
|**[[2.4.4.3 Scope Attribute]]**|Accessibility|`col` vs `row`|The `scope` attribute tells screen readers "I am a header for this column".|
|**[[2.4.4.4 Placement]]**|Top or Side|First row/First col|Headers are usually in the first row (cols) or first column (rows).|
|**[[2.4.4.5 Abbr Attribute]]**|Accessibility|Short name|You can provide a shorter version of the header for screen readers.|
