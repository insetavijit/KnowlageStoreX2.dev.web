|---|---|---|---|
|**2.4.5 Simple Table Structure**|Demonstrate basic anatomy|Code pattern, Grid visualization|The most basic valid table consists of rows and cells.|
|**[[2.4.5.1 The Pattern]]**|Code|`table > tr > td`|It's a box (table) of shelves (tr) holding items (td).|
|**[[2.4.5.2 Equal Cells]]**|Rule|Grid consistency|Each row should have the same number of cells (unless spanning).|
|**[[2.4.5.3 First Row Headers]]**|Best Practice|`<th>`|The first `<tr>` usually contains `<th>` elements.|
|**[[2.4.5.4 Border Collapse]]**|Styling|CSS|By default, cells have gaps; `border-collapse: collapse` removes them.|
|**[[2.4.5.5 Caption]]**|Labeling|`<caption>`|A proper simple table includes a caption to describe it.|
|**[[2.4.5.6 Visual Debugging]]**|DevTools|Borders|Adding `border="1"` (temporarily) helps you see the grid while building.|
