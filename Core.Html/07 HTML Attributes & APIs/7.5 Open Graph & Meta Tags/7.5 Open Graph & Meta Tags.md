| Topic                                        | Focus & Purpose (Researched & Evaluated)                                                                                                                                                                                           |
| -------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **[[7.5.1 Meta Tags Overview]]**             | Introduce **meta elements**: metadata in document head, name/content pairs, http-equiv, charset, and the role of meta tags in HTML documents.                                                                                     |
| **[[7.5.2 Open Graph Protocol]]**            | Explain **Open Graph tags**: Facebook's metadata protocol, og:title, og:description, og:image, og:url; controlling how pages appear when shared on social media.                                                                  |
| **[[7.5.3 Twitter Card Tags]]**             | Cover **Twitter card metadata**: twitter:card, twitter:title, twitter:description, twitter:image; card types (summary, summary_large_image), and Twitter-specific sharing optimization.                                           |
| **[[7.5.4 Description Meta Tag]]**           | Detail **meta description**: search result snippets, character limits, compelling descriptions, keyword inclusion, and description best practices.                                                                                |
| **[[7.5.5 Viewport Meta Tag]]**              | Cover the **viewport meta tag** (review): width=device-width, initial-scale, maximum-scale, user-scalable, and mobile responsive foundation.                                                                                      |
| **[[7.5.6 Charset Meta Tag]]**               | Review **character encoding declaration**: UTF-8 recommendation, meta charset placement, encoding issues without proper declaration, and international character support.                                                         |
| **[[7.5.7 Robots Meta Tag]]**                | Explain **robots directives**: index/noindex, follow/nofollow, noarchive, nosnippet; controlling search engine behavior per page.                                                                                                 |
| **[[7.5.8 Theme Color Meta Tag]]**           | Cover **theme-color meta tag**: browser UI color customization, PWA toolbar color, media queries for dark mode, and mobile browser styling.                                                                                      |
| **[[7.5.9 Canonical Link]]**                 | Explain **rel="canonical"**: duplicate content handling, specifying preferred URL, SEO implications, and canonical usage patterns.                                                                                                |
| **[[7.5.10 Other Essential Meta]]**          | Cover **additional meta tags**: author, generator, application-name, format-detection, referrer, and less common but useful meta declarations.                                                                                    |
