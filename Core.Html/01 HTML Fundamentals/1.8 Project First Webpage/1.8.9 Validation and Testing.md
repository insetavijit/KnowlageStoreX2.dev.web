|---|---|---|---|
|**1.8.9 Validation and Testing**|Perform quality assurance|W3C Validator, Browser check|Don't ship it until you prove it works.|
|**[[1.8.9.1 W3C Validator]]**|Tool|validator.w3.org|Upload your file to check for syntax errors.|
|**[[1.8.9.2 Common Errors]]**|Debugging|Unclosed tags, nesting issues|Fix the errors the validator finds (like unclosed `<div>`).|
|**[[1.8.9.3 Browser Testing]]**|Cross-browser|Chrome/Firefox/Edge|Open the file in at least two different browsers.|
|**[[1.8.9.4 Link Checking]]**|QA|Click everything|Manually click every single link to ensure it goes where expected.|
|**[[1.8.9.5 Spell Check]]**|Content|Typos|Read your text; code can be perfect but typos look unprofessional.|
|**[[1.8.9.6 Mobile Check]]**|Responsiveness|Resize window|Shrink your browser window to see if horizontal scrollbars appear.|
