|---|---|---|---|
|**1.6.2 The href Attribute**|Explain the link destination|Hypertext Reference, URL, URI|`href` stands for Hypertext Reference; it's where the link goes.|
|**[[1.6.2.1 Required Attribute]]**|Validation|No href = No link|Without `href`, an anchor is just a span (a placeholder).|
|**[[1.6.2.2 Valid Values]]**|Destinations|URL, Path, Fragment, Script (avoid)|It can take an absolute URL, relative path, or hash ID.|
|**[[1.6.2.3 Empty Values]]**|Antipattern|`href=""` or `href="#"`|Using `#` refreshes the page or jumps to top; avoid for buttons.|
|**[[1.6.2.4 Javascript URLs]]**|Legacy pattern|`javascript:void(0)`|Avoid `javascript:` in href; use event listeners instead.|
|**[[1.6.2.5 Protocol Relative]]**|Advanced|`//example.com`|Inherits the protocol (http/https) of the current page.|
