|**1.7.2 The src Attribute**|Explain image location|Absolute, Relative, Data URIs|The source attribute tells the browser where to find the image file.|
|**[[1.7.2.1 Relative Paths]]**|Local assets|`./images/pic.jpg`|Most common way to link images within your project folder.|
|**[[1.7.2.2 Absolute URLs]]**|External assets|`https://site.com/pic.jpg`|Hotlinking images from other servers (be careful of bandwidth theft).|
|**[[1.7.2.3 Data URIs]]**|Base64|`data:image/png;base64...`|Embedding the image data directly into the HTML code.|
|**[[1.7.2.4 Broken Images]]**|Error state|404 handling|If the src is wrong, the browser shows a broken image icon and the alt text.|
|**[[1.7.2.5 Protocols]]**|Mixing|HTTP vs HTTPS|Don't load HTTP images on an HTTPS site (Mixed Content warning).|
|**[[1.7.2.6 Caching]]**|Performance|Browser cache|The browser downloads the src once and reuses it.|
