|---|---|---|---|
|**1.7.9 Decorative vs Informative**|Distinguish image purpose|Accessibility choice, Alt text strategy|Not every image needs to be described.|
|**[[1.7.9.1 Informative Images]]**|Meaningful|Graphs, Product photos|Images that add info to the content must have descriptive alt text.|
|**[[1.7.9.2 Decorative Images]]**|Styling|Dividers, Backgrounds|Images purely for looks should have `alt=""`.|
|**[[1.7.9.3 Functional Images]]**|Actionable|Icons in buttons|Images inside buttons should describe the *action* (e.g., "Search"), not the look.|
|**[[1.7.9.4 Complex Images]]**|Detailed|Graphs, Infographics|Complex images need `alt` *plus* a long description nearby.|
|**[[1.7.9.5 Context Matters]]**|Nuance|Redundancy|If the caption says it, the alt text shouldn't repeat it.|
|**[[1.7.9.6 CSS Backgrounds]]**|Alternative|CSS properties|True decoration (patterns) often belongs in CSS, not HTML.|
