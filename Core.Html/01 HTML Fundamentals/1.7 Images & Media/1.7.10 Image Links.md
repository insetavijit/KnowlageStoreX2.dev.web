|**1.7.10 Image Links**|Explain images as link content|Nesting, Alt text implication, Borders|An image can be a button to another place.|
|**[[1.7.10.1 Syntax]]**|Nesting|`<a><img /></a>`|Wrap the image in an anchor tag.|
|**[[1.7.10.2 Alt Text Role]]**|Labeling|Link function|For linked images, the alt text becomes the link text (e.g., "Go to Home").|
|**[[1.7.10.3 Visual Cues]]**|Borders|Legacy styling|Old browsers put blue borders around linked images; remove with CSS.|
|**[[1.7.10.4 Image Maps]]**|Legacy feature|`<map>`, `<area>`|Defining clickable hot spots on a single image (rarely used now).|
|**[[1.7.10.5 Buttons vs Links]]**|UX|Action vs Navigation|Use `<button>` with an image icon for actions; `<a>` for navigation.|
|**[[1.7.10.6 Hover States]]**|Feedback|Opacity/Transform|Add CSS hover effects so users know the image is clickable.|
