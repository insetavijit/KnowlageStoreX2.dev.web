|**1.7.5 Width and Height Attributes**|Explain dimension attributes|CLS (Cumulative Layout Shift), Intrinsic size|Always tell the browser the size of the image to prevent layout jumping.|
|**[[1.7.5.1 Layout Shift]]**|The problem|Janky loading|Without dimensions, the text jumps when the image finally loads.|
|**[[1.7.5.2 Syntax]]**|Unitless integers|`width="500" height="300"`|Don't use `px`; just use numbers.|
|**[[1.7.5.3 Aspect Ratio Check]]**|Browser calculation|Reserving space|The browser uses these numbers to reserve a hole of the correct shape.|
|**[[1.7.5.4 Responsive CSS]]**|Scaling|`max-width: 100%; height: auto;`|CSS makes the image fluid; HTML attributes define the ratio.|
|**[[1.7.5.5 Performance]]**|Rendering speed|Less layout work|Pre-defining size helps the browser calculate layout faster.|
|**[[1.7.5.6 Distinction]]**|Attr vs CSS|Presentation separation|Use attributes for ratio reservation, CSS for actual display size.|
