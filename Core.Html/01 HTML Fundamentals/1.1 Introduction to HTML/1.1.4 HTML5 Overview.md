|**1.1.4 HTML5 Overview**|Summarize enhancements HTML5 brought to the web|Semantics, Multimedia, APIs, Offline storage|HTML5 introduced native apps, media, and semantics to the web.|
|**[[1.1.4.1 New Semantic Elements]]**|Structure vs generic divs|`<nav>`, `<section>`, `<article>`, `<header>`, `<footer>`|HTML5 added elements that describe what content *is*.|
|**[[1.1.4.2 Multimedia Support]]**|Native audio and video without plugins|`<audio>`, `<video>`, `<source>`, removal of Flash|HTML5 enabled native playback of media in the browser.|
|**[[1.1.4.3 Form Enhancements]]**|New input types and validation|Inputs: `date`, `email`; Attributes: `required`, `pattern`|HTML5 forms handle validation and specialized inputs natively.|
|**[[1.1.4.4 New APIs]]**|Capabilities beyond simple documents|Geolocation, Drag & Drop, LocalStorage, Web Workers|HTML5 APIs gave web pages the power of desktop applications.|
|**[[1.1.4.5 Removed Elements]]**|Cleaning up the language|`<font>`, `<center>`, `<frame>`, `<big>` (moved to CSS)|Presentational tags were removed in favor of CSS.|
|**[[1.1.4.6 Doctype & Charset]]**|Simplifying the boilerplate|`<!DOCTYPE html>`, `<meta charset="UTF-8">`|HTML5 dramatically simplified the required document header.|
