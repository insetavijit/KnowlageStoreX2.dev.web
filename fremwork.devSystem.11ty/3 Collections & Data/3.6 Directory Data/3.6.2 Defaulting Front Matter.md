# Defaulting Front Matter: DRY Configuration

The most common use of directory data is to **set default front matter** for an entire content section. This eliminates repetition and ensures consistency across related files.

---

## The Fundamental Principle: Define Once, Apply Everywhere

If every blog post needs `layout: post.njk`, writing it in 200 files is wasteful and error-prone. Directory data lets you define it once.

---

## Common Default Patterns

**Layout:**
```json
{
  "layout": "post.njk"
}
```

**Tags:**
```json
{
  "tags": ["blog"]
}
```

**Combined Configuration:**
```json
{
  "layout": "post.njk",
  "tags": ["blog"],
  "author": "Editorial Team",
  "showComments": true,
  "eleventyNavigation": {
    "parent": "Blog"
  }
}
```

---

## Permalink Patterns

Set a consistent URL structure for all files in a folder:

```json
{
  "permalink": "/blog/{{ page.fileSlug }}/index.html"
}
```

Or with date-based URLs:

```json
{
  "permalink": "/blog/{{ page.date | date: '%Y/%m' }}/{{ page.fileSlug }}/"
}
```

All files in the folder will follow this pattern automatically.

---

## Multiple Tags (Merging)

With deep merge enabled, tags combine across levels:

**`posts/posts.json`:**
```json
{
  "tags": ["blog"]
}
```

**`posts/hello.md`:**
```yaml
---
tags:
  - featured
---
```

**Result:**
```javascript
{ tags: ["blog", "featured"] }
```

The post appears in both `collections.blog` and `collections.featured`.

---

## SEO Defaults

Set default meta tags that content can override:

```json
{
  "meta": {
    "og:type": "article",
    "twitter:card": "summary_large_image"
  }
}
```

Individual posts can then add or override specific properties.

---

## Opt-Out Pattern

When most files need a default but some don't:

**Directory Default:**
```json
{
  "showSidebar": true
}
```

**Specific File Override:**
```yaml
---
showSidebar: false
---
```

The directory provides the default; exceptional files opt out explicitly.

---

## Common Pitfalls and Solutions

**Problem:** All posts use the same layout but one shouldn't.

**Solution:** Set the default in directory data, override in the specific file's front matter.

---

**Problem:** Tags don't merge (replace instead).

**Solution:** Enable `setDataDeepMerge(true)` in your Eleventy config.
